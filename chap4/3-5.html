input의 속성들

            

            input type="text" 한 줄 텍스트 입력하는 창
            input type="password" 암호 입력을 위한 텍스트 입력 창
            input type="button" 단순 버튼
            input type="submit" 웹 서버로 데이터를 전송함
            input type="reset" 입력된 폼 데이터를 초기화 시키는 버튼
            input type="image" 이미지 버튼
            input type="check" 체크박스와 라디오 버튼
            input type="color" 색상입력을 쉽게하는 다이어로그
            input type="file" 컴퓨터 파일을 선택하는 폼요소
